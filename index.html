Belajar Git & GitHub sambil Langsung Praktek

Materi ini dirancang untuk workshop: setiap modul berisi konsep singkat, contoh perintah yang bisa langsung disalin, dan latihan nyata. Website ini sendiri adalah contoh proyek statis yang akan kalian git init, commit, lalu push ke GitHub — jadi kalian belajar Git sambil benar-benar memakainya.

Halaman ini adalah versi lengkap dari sesi Slide Presentasi 60 menit yang dibawakan langsung — kalau kalian lupa satu-dua bagian setelah sesi selesai, atau baru mau mulai belajar sendiri, tiap modul di sini membahas hal yang sama dengan lebih detail dan contoh perintah yang bisa langsung dipraktekkan.

Alur Belajar

Ikuti modul secara berurutan. Modul 1–3 membangun fondasi Git lokal, modul 4 mengenalkan branching, modul 5–6 menyambungkan ke GitHub dan kolaborasi tim, dan modul 7 merangkum kebiasaan baik yang dipakai tim profesional.

Interaktif

Prasyarat

  • Laptop dengan akses terminal (Terminal / PowerShell / Git Bash).
  • Akun GitHub (buat dulu jika belum punya).
  • Text editor apa saja (disarankan VS Code).
  • Tidak perlu pengalaman Git sebelumnya.
