Instalasi & Konfigurasi Git

Pasang Git di komputer kalian, lalu kenalkan diri kalian ke Git supaya setiap commit tercatat atas nama yang benar.

Di modul ini
  1. Instalasi per sistem operasi
  2. Verifikasi instalasi
  3. Konfigurasi identitas
  4. Cek & kelola konfigurasi

1. Instalasi Git

Windows

Unduh installer dari git-scm.com, jalankan, lalu ikuti wizard (opsi default sudah aman untuk pemula). Installer ini juga menyertakan Git Bash, terminal yang direkomendasikan untuk workshop ini.

macOS

Paling gampang lewat Homebrew:

bash 
brew install git

Atau, macOS biasanya sudah menyertakan Git bawaan Xcode Command Line Tools — coba jalankan git --version, kalau belum ada, macOS akan menawarkan instalasi otomatis.

Linux (Debian/Ubuntu)

bash 
sudo apt update
sudo apt install git

Linux (Fedora/RHEL)

bash 
sudo dnf install git

2. Verifikasi Instalasi

Buka terminal (Git Bash di Windows, Terminal di macOS/Linux) lalu jalankan:

bash 
git --version

Kalau muncul versi seperti git version 2.4x.x, instalasi berhasil.

3. Konfigurasi Identitas

Setiap commit di Git membawa informasi siapa yang membuatnya. Sebelum commit pertama, atur nama dan email kalian secara global (berlaku untuk semua repo di komputer ini):

bash 
git config --global user.name "Nama Kamu"
git config --global user.email "email@kamu.com"
⚠️
Samakan dengan email GitHub

Gunakan email yang sama dengan akun GitHub kalian (atau yang terdaftar di GitHub), supaya commit kalian nanti otomatis terhubung ke profil GitHub — bukan tampil sebagai "unknown author".

Opsional, tapi sering berguna — atur editor default untuk pesan commit dan nama branch utama:

bash 
git config --global core.editor "code --wait"   # pakai VS Code
git config --global init.defaultBranch main     # nama branch utama = "main"

4. Cek & Kelola Konfigurasi

Lihat semua konfigurasi yang aktif:

bash 
git config --list

Atau cek satu nilai spesifik:

bash 
git config user.name
git config user.email

Konfigurasi Git tersimpan berlapis, dari yang paling luas ke paling spesifik:

LevelFlagBerlaku untuk
System--systemSemua user di komputer ini
Global--globalSemua repo milik user kalian
Local(tanpa flag, di dalam repo)Hanya repo yang sedang dibuka
✅
Checkpoint

Sebelum lanjut ke Modul 3, pastikan git --version menampilkan versi Git, dan git config user.name / git config user.email menampilkan identitas kalian.
